
# 🎬 YouTube‑Clone

A modern and responsive YouTube-inspired web application built with **React**, **Vite**, and **TailwindCSS**.

The project allows users to browse videos, search content, filter by categories, and watch videos through an intuitive dark-themed interface powered by the YouTube Data API.



## 🚀 Features

- **Responsive UI** – adapts to desktop, tablet & mobile.
- **Sticky sidebar** with project‑wide dark scrollbar styling.
- **Hover‑play preview** – auto‑play on mouse‑enter, pause on leave.
- **Category navigation** – Trendler, Müzik, Canlı, Oyun, Spor, …
- **Search & filter** – URL query support (`?category=…`).
- **React‑Player** – YouTube playback with native controls.
- **Tailwind‑CSS** theming – custom `--color‑black` & `--color‑grey`.
- **Modular codebase** – structured components and pages.
